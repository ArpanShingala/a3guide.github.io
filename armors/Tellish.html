Tellish Set
Tellish originally means "strong willed". This armor is different from other armors that can be bought from shops or attained via barter trade. There is an unsaid rule among the archers that only those who are strong can wear this set. Therefore, this armor set is also known as the initiation armor for all the crossbow users.

Grade: 6
Class: Archer
Set Type: Crossbow Archer

Set Requirements

Level: 0
Strength: 342
Dexterity: 172
